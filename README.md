# Netflix Data Analysis

This repository contains Python and R code for analyzing Netflix shows and movies data.

## Requirements

- Python 3.x
- R 4.x
- Libraries:
  - Python: `pandas`, `os`, `matplotlib`, `seaborn`
  - R: `ggplot2`, `dplyr`

## Setup

1. Ensure you have Python and R installed on your system.
2. Install required Python libraries: `pandas`, `os`, `matplotlib`, `seaborn`
3. Install required R libraries: `ggplot2`, `dplyr`

## Running the Code

### Python Analysis
1. Open the python script `netflix_analysis.py`
2. Run the script in the required python environment

### R Analysis
1. Open the R script `netflix_visualization.R`
2. Run the script in RStudio

## File Descriptions

- `netflix_data.csv`: The dataset to be renamed to `Netflix_shows_movies.csv` and used for analysis 
- `Netflix_cleaned.csv`: Dataset for R visualization
- `netflix_analysis.py`: Python script for renaming file, data cleaning, exploration, and visualization
- `netflix_visualization.R`: R script for creating the ratings distribution chart

## Results

The analysis includes:
1. Data cleaning and handling of missing values
2. Exploratory data analysis
3. Visualization of most watched genres
4. Visualization of ratings distribution

Refer to the output and comments in `netflix_analysis.py` and `netflix_visualization.R` for detailed insights.
